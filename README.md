# 北京航空航天大学教育基金会 H5

北京航空航天大学教育基金会移动端 H5 网站。

## 项目介绍

基于 Vue3 + Vant 开发的移动端静态体验版，包含基金会核心功能展示。

## 功能模块

- ✅ **首页** - 轮播图、快捷入口、捐赠项目、文创产品、校友活动、基金会动态
- 🚧 **捐赠功能** - 展示捐赠项目列表（静态数据）
- 🚧 **文创商城** - 文创产品展示（静态数据）
- 🚧 **校友活动** - 活动列表展示
- 🚧 **基金会动态** - 新闻公告
- 🚧 **个人中心** - 我的页面

## 技术栈

- Vue3 (CDN)
- Vant 4 (CDN)
- HTML5
- CSS3
- JavaScript

## 开发计划

### 第一阶段：静态体验版 ✅
- 首页展示

### 第二阶段：对接后端 API

- 在线捐赠支付
